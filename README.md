# MERN Blog App Client 

*Blog app with CRUD operations.* 
*User can login/register using credentials.*
*Update user account.*
*Delete user account.*
*Upload profile picture.*
*Create a blog post.*
*Edit the post.*
*Delete the post.*


## Tech Stack

**Client:** React, Context API

**Server:** Node, Express

**Database:** MongoDB
